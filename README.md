A high-performance, visually stunning portfolio designed to showcase the intersection of aesthetic branding and functional full-stack development. This project focuses on premium user experience, smooth animations, and optimized loading speeds.

---

## ✨ Features

- **🚀 Performance Optimized**: Optimized critical path rendering, lazy-loaded assets, and streamlined JavaScript for near-instant starting times.
- **🪄 Premium UI/UX**: A dark-themed, glassmorphic design system using a curated "Midnight Blossom" palette.
- **✨ Fluid Animations**: Custom cursor interactions, 3D tilt effects on project cards, and staggered scroll reveals using Intersection Observer.
- **📱 Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.
- **🛠️ Full-Stack Showcased**: Dedicated sections for design work, mobile applications, and web development.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System)
- **Interactions**: Vanilla JavaScript (ES6+), CSS Keyframes
- **Icons**: SVG Path Graphics
- **Fonts**: Playfair Display (Serif) & Inter (Sans-serif)
- **Optimization**: Lazy Loading, Critical Path CSS, Debounced Animations

## 🚀 Performance Highlights

The latest version includes significant performance updates:
- **Reduced Time to Interactive (TTI)**: By optimizing the loader logic and removing artificial delays.
- **Improved LCP**: By implementing lazy loading and explicit image dimensions to prevent layout shifts.
- **Instant Loader**: Critical CSS is inlined to ensure the UI feels responsive from the very first byte.
